竞技场步骤
1、连接WebSocket
2、获取角色信息 (role_getroleinfo)
3、提取登录时间 (statistics['last:login:time'])
4、本地计算随机数种子（使用XOR算法）：
const XOR_A = 2118920861;
const XOR_B = 797788954;
const XOR_C = 1513922175;

let seed = lastLoginTime | 0;
seed ^= XOR_A;
seed = ((seed << 16) | (seed >>> 16)) >>> 0;
seed ^= XOR_B;
seed ^= XOR_C;
const randomSeed = seed >>> 0;

5、发送随机数种子 (system_custom with key: 'randomSeed')
6、开始竞技场 (arena_startarea)
7、获取目标并战斗 (arena_getareatarget → fight_startarenarena)
